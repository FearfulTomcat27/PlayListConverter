# 这是个什么？

这是导出汽水音乐歌单的项目

## 开始

### 下载

克隆这个仓库:

```
git clone https://github.com/FearfulTomcat27/PlayListConverter
cd PlayListConverter
```

用pip下载依赖:

```
pip install -r requirements.txt
```

### 运行

Bash或者Cmd(把{url}换成你想要导出歌单的地址)

``` 
python main.py -u {url}
```

**Or**

```
python main.py
```

**Pycharm:**

运行``` main.py``` 并填写url
